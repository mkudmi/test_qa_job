Инструкции для запусков тестов:

1. SMOKE тесты. Проверка основных запросов.

bash:
python3 -m pytest -v -s -m smoke

2. PERFORMANCE тесты. Проверка скорости запросов.

bash:
python3 -m pytest -v -s -m performance

3. NEGATIVE тесты. Проверка на отправку некорректных данных.

bash:
python3 -m pytest -v -s -m negative

4. SECURITY тесты. Проверка на авторизацию (но ее там вроде и нет, тем не менне)

bash:
python3 -m pytest -v -s -m security
